# Projekt SQL - Engeto

Tento projekt analyzuje růst HDP, mezd a cen potravin v České republice od roku 2006.  
Data byla získána z databáze a zpracována pomocí SQL dotazů.  

## Obsah
- `final_results.csv` - soubor obsahující výsledky analýzy  

## Použité technologie
- PostgreSQL  
- DBeaver  
- GitHub
